Лекарства¶

Открывает новое окно со списком лекарств, имеющихся в наличии:

Здесь содержится необходимая табличная информация о лекарственных средствах, а также кнопки:

Добавить:открывает форму для добавления нового лекарственного средства в таблицу.
Редактировать:открывает форму для редактирования выбранного лекарственного средства (необходимо выделить строку нажатием левой кнопки мыши).
Удалить:удаляет ЛС из таблицы (также необходимо выделить строку левой кнопкой мыши).

Файл ‣ Выход¶

Выход из программы.

Справочники ‣ Типы лекарств¶

Открывает новое окно с типами ЛС. Это одна из характеристик, присваиваемых во время добавленияредактирования ЛС.

Справочники ‣ Поставщики¶

Открывает справочник со сведениями о доступных поставщиках.

Справочники ‣ Запросы на поставку¶

Открывает окно со списком запросов на поставку ЛС:

Добавить:открывает форму для добавления нового заказа в таблицу.
Редактировать:открывает форму для редактирования выбранного заказа (необходимо выделить строку нажатием левой кнопки мыши).
Удалить:удаляет заказ из таблицы (также необходимо выделить строку левой кнопкой мыши).

Справочники ‣ Книга учета¶

Открывает окно книги учета, в которую вносятся удовлетворенные клиенты.

Справочники ‣ Рецепты¶

Открывает окно со списком всех зарегистрированных рецептов:

Добавить:открывает форму для добавления нового рецепта в таблицу.
Редактировать:открывает форму для редактирования выбранного рецепта (необходимо выделить строку нажатием левой кнопки мыши).
Удалить:удаляет рецепт из таблицы (также необходимо выделить строку левой кнопкой мыши).

Справочники ‣ Заказы в обработке¶

Открывает список обрабатываемых клиентских заказов. Заказы имеют пометки: «Исполнен», в случае удовлетворения заказа; «В процессе», если заказ еще выполняется (ожидание поставки и т.п.); и «Не исполнен», в случае отсутствия ЛС как на складе аптеки, так и у поставщиков.
